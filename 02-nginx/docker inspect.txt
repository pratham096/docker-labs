docker run -d --name demo -p 8081:80 nginx

1.docker run → Start a new container.
2.-d → Run in background (detached mode).
3.--name demo → Name the container demo.
4.-p 8081:80 → Map port 8081 on your host → port 80 in the container (Nginx default).
5.nginx → Use the latest Nginx image from Docker Hub.


docker inspect demo | jq '.[0].NetworkSettings.IPAddress'
1.docker inspect demo → Shows all container details in JSON (network, volumes, configs, etc.).
2.| jq ... → Pipes the JSON to jq (a JSON parser) to extract the value.
3. .[0].NetworkSettings.IPAddress → Prints the container’s internal Docker network IP (e.g., "172.17.0.2").


docker rm -f demo
1. docker rm → Remove a container.
2. -f → Force stop & remove it in one step (without having to stop it first).
